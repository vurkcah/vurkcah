Реализованная функциональность:
1. Просмотр всех заявок на субсидии
2. Обеспечение невозможности удаления и изменения заявок без внимания других обладателей базы данных 
3. Возможность использовать опыт отказа для корректировки будущей заявки

Особенность проекта в следующем:
1. Поиск по параметрам в блокчейне
2. Упрощение процедуры получения субсидий

Основной стек технологий:
HTML, CSS, JavaScript, TypeScript
